# Universo Maker - Frontend

## Tecnologias
- React 18
- Vite
- Tailwind CSS
- Framer Motion (animações)
- React Router DOM
- React Hook Form
- Axios
- Lucide React (ícones)

## Paleta de Cores
- **Primary**: #024059 (Azul escuro)
- **Secondary**: #33A67C (Verde)
- **Accent**: #F2AE30 (Amarelo)
- **Light**: #F2EDE4 (Bege claro)
- **Danger**: #F24822 (Vermelho)

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Funcionalidades

- ✅ Login responsivo com animações
- ✅ Dashboard com estatísticas
- ✅ Gestão de alunos
- ✅ Gestão de PEI
- ✅ Layout responsivo com sidebar
- ✅ Animações suaves
- ✅ Design moderno e infantil
- ✅ Integração com API backend

## Estrutura

```
src/
├── components/     # Componentes reutilizáveis
├── pages/         # Páginas da aplicação
├── hooks/         # Hooks customizados
├── services/      # Configuração da API
└── assets/        # Imagens e recursos
```

Acesse: http://localhost:3000
